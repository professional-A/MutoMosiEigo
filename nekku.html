第1章

流体の基本性質

密度・比重量・比体積 基本
ρ = m / V　　　[kg/m³]
γ = W / V = ρg　[N/m³]（比重量）
v = V / m = 1/ρ　[m³/kg]（比体積）
s = ρ / ρw　（比重、ρw = 1000 kg/m³）
記号説明単位
ρ密度kg/m³
m質量kg
V体積m³
γ比重量（単位体積あたりの重量）N/m³
g重力加速度 ≈ 9.81m/s²
s比重（無次元）—
💡 水の密度 ρw = 1000 kg/m³、水銀 ρHg = 13600 kg/m³、海水 ≈ 1025 kg/m³
ニュートンの粘性法則 重要
τ = μ (du/dy)　　[Pa = N/m²]
一定速度勾配の場合：τ = μ (U / Y)
せん断力：F = τA = μA (U / Y)
記号説明単位
τせん断応力Pa
μ粘性係数（粘度）Pa·s
du/dy速度勾配（せん断速度）1/s
U板の速度m/s
Y板間の距離（すき間幅）m
Fせん断力N
A板の面積m²

導出のポイント

粘性流体は隣り合う流体層がずれると抵抗力が生じる。速度差 U がある間隔 Y の平板間では速度は線形変化 → 勾配 = U/Y。

例題（両面せん断）

板が上下の液体層（隙間 Y₁, Y₂）の中にある場合：
F = μAU (1/Y₁ + 1/Y₂)
引く力が最小になる板の位置：x = h/2（中央）

動粘度
ν = μ / ρ　　[m²/s]

例題（傾斜面を滑落する物体の最大速度）

油膜（厚さ δ）の上を質量 m の物体が傾斜角 θ の面を滑落するとき、重力とせん断力がつり合う最大速度：
Umax = (δ · m · g · sinθ) / (μ · A)
※ mg sinθ = μA(Umax/δ) のつり合いから導出

第2章①

静水圧・パスカルの原理

圧力の定義
P = F / A　　[Pa = N/m²]
円形断面の場合：P = 4F / (πd²)
パスカルの原理 重要
F₁/A₁ = F₂/A₂ = P　（圧力は等しく伝わる）
直径表記：F₂ = (d₂/d₁)² × F₁

導出のポイント

密閉容器内の静止流体に加えた圧力は、あらゆる方向に等しく伝わる（パスカルの原理）。面積比だけ力が増幅される → 油圧ジャッキの原理。

例題（レバー付き油圧装置）

レバーでの力変換：f × L₁ = F × L₂ でまず小ピストンの力 F を求め、
次に P = F/A でシリンダ圧力を求め、大ピストンに F₂ = P × A₂ を適用する。

静止流体内の圧力（静水圧公式） 重要
P = P₀ + ρgh　　[Pa]　…(2.9)
ゲージ圧：p = P − P₀ = ρgh　[Pa]　…(2.10)
圧力ヘッド：h = p / (ρg)　[m]　…(2.11)
記号説明単位
P絶対圧力（真空 = 0 基準）Pa
P₀大気圧（≈ 101.3 kPa）Pa
pゲージ圧（大気圧基準）Pa
ρ液体の密度kg/m³
h液面からの深さm

導出のポイント

深さ h の微小要素に作用する力のつり合いを取ると、深さが増すほど上の液体の重さが加わる。結果として p = ρgh（深さに比例）。

例題

水面下 5m のゲージ圧：p = 1000 × 9.81 × 5 = 49.1 kPa
絶対圧：P = 101.3 + 49.1 = 150.4 kPa
水銀換算：760 mmHg = 13600 × 9.81 × 0.760 = 101.4 kPa

第2章②

液柱計（マノメータ）

💡 基本原理：連通した静止液体では、同一水平面上の圧力は等しい。基準面（A面）での左右の圧力を等値して解く。
単純マノメータ（液柱計）
P = P₀ + ρ'gh − ρgy　…(2.13)
記号説明
ρ'細管内液体の密度
ρ容器内液体の密度
h細管内液面の高さ
y基準面から容器液面までの距離

導出

基準面 A での左右圧力を等値：
左：P_A = P + ρgy　右：P_A = P₀ + ρ'gh
①=② → P = P₀ + ρ'gh − ρgy

U字管マノメータ 頻出
P₁ − P₂ = ρ₂gy₂ + ρ'gh − ρ₁gy₁　…(2.14)
同液体（ρ₁=ρ₂=ρ）の場合：
P₁ − P₂ = ρg(y₂ − y₁) + ρ'gh

導出

基準面 A の圧力等値：
左：P_A = P₁ + ρ₁gy₁
右：P_A = P₂ + ρ₂gy₂ + ρ'gh

逆U字管マノメータ
P₁ − P₂ = ρ₁gy₁ − ρ'gh − ρ₂gy₂　…(2.15)

導出

上部管の流体密度が小さい（ρ' < ρ）。
基準面 A の圧力等値（符号が逆になることに注意）。

例題（逆U字管、水 ρ=1000、油 ρ'=850）

P_B − P_A = (ρ − ρ')gh = (1000 − 850) × 9.81 × 0.075 = 110 Pa

第2章③

平面板・曲面板に作用する力

平面板に作用する全静水圧力 重要
F = ρg hG A　　[N]　…(2.21)
= ρg yG sinθ · A　（傾斜面表記）
記号説明単位
hG水面から平面板の図心 G までの鉛直深さm
A平面板の面積m²
yG基準点 O から図心 G までの斜面に沿った距離m
θ平面板と水平面のなす角°

導出のポイント

圧力は深さに比例する（p = ρgh）。板全体の合力 = 板の各微小面積に作用する圧力の積分 → 図心での圧力 × 全面積。

傾斜角と深さの関係：hG = yG sinθ　→　yG = hG / sinθ
圧力中心（力の作用点） 重要
yC = IG / (yG · A) + yG　…(2.25)
記号説明
yC基準点 O から圧力中心 C までの斜面距離
IG図心 G まわりの断面 2 次モーメント
圧力中心は常に図心より深い側（yC > yG）。圧力が深くなるほど大きいため。
断面 2 次モーメント IG
円（直径 d）：IG = πd⁴ / 64　→　yC = d² / (16yG) + yG
長方形（幅 a, 高さ b）：IG = ab³ / 12　→　yC = b² / (12yG) + yG

例題（傾斜した円板、θ = 36°、d = 1.8m、h = 0.75m）

hG = 0.75 + (1.8/2)×sin36° = 1.279 m
yG = 1.279 / sin36° = 2.176 m
F = 1000 × 9.81 × 1.279 × π×1.8²/4 = 31.9 kN
yC = 1.8² / (16×2.176) + 2.176 = 2.269 m

曲面板に作用する力 重要
Fx = ρg z̄ Ax　（水平分力）　…(2.26)
Fy = ρg V　　（鉛直分力）　…(2.27)
F = √(Fx² + Fy²)　…(2.28)
α = tan⁻¹(Fy / Fx)　（合力の方向角）　…(2.29)
記号説明
Ax曲面を水平方向に投影した面積
z̄水平投影面の図心深さ
V曲面板の上に乗っている液体の体積

導出のポイント

水平分力 Fx：曲面を横から見た「投影平板」への静水圧と同じ。
鉛直分力 Fy：曲面板の上の液体の重量（重さ = 浮力の反作用）。

例題（1/4 円筒、半径 r = 1.2m、幅 b = 4m）

Fx = ρg(r/2) × br = 1000×9.81×0.6×4×1.2 = 28.25 kN
Fy = ρg(πr²/4)×b = 1000×9.81×π×1.44/4×4 = 44.38 kN
F = √(28.25² + 44.38²) = 52.6 kN、α = 57.5°

第2章④

浮力（アルキメデスの原理）

浮力 重要
F = ρ流体 · g · V排除　…(2.30)
記号説明
ρ流体流体（液体）の密度（物体の密度ではない！）
V排除物体が液体中に沈んでいる部分の体積

アルキメデスの原理

液体中に沈んだ物体は、排除した液体の重量に等しい上向きの力（浮力）を受ける。

重量＝浮力のつり合い式

物体の重量：mg = ρ物体 V g
浮力：F = ρ流体 V沈 g
つり合い：ρ物体 V g = ρ流体 (V − V1) g
→ V = ρ流体 V1 / (ρ流体 − ρ物体)　（氷山の全体積）

直方体が水面に浮く場合の沈む深さ

mg = ρ水 g a b h → h = m / (ρ水 a b)

第3章

連続の式（質量保存）

連続の式（定常非圧縮流れ） 重要
A₁u₁ = A₂u₂ = Q = 一定　…(3.7)
直径表記：u₂ = (d₁/d₂)² × u₁
記号説明単位
A断面積m²
u断面平均流速m/s
Q体積流量m³/s
d内径m

導出のポイント

非圧縮性（密度一定）の定常流では、単位時間に断面を通過する体積は変化しない（質量保存）。断面積が小さくなれば流速が大きくなる。

例題（管径が変わる場合）

d₁ = 80mm → d₂ = 40mm、u₁ = 6 m/s のとき：
u₂ = (80/40)² × 6 = 4 × 6 = 24 m/s

流量・質量流量
Q = Au = (πd²/4) u　[m³/s]
G = ρQ = ρAu　　　　[kg/s]（質量流量）
ua = Q / A　　　　　[m/s]（平均流速）　…(3.8)

分岐管（Y字管）の連続の式

Q₁ = Q₂ + Q₃
等量分岐（Q₂ = Q₃）の場合：u₂ = (1/2)(d₁/d₂)² u₁
タンク充填時間：T = (D/d)² × (h/u)

速度分布と平均流速 積分
【円管・放物線分布】
u = umax (1 − r²/R²)
平均流速：ua = umax / 2
【平板間・放物線分布】
v = 2vmax (y/Y)(1 − y/Y)
流量（幅 b）：Q = bvmaxY / 3

放物線（円管）の積分

微小環状面積 dA = 2πr dr に速度 u を掛けて積分：
Q = ∫₀ᴿ 2πr·umax(1−r²/R²)dr = πR²umax/2
→ ua = Q/A = umax/2

第4章

ベルヌーイの定理とその応用

ベルヌーイの定理（定常非圧縮流れ） 最重要
p + ρu²/2 + ρgz = 一定　[Pa]　…(4.8)
2点間：p₁ + ρu₁²/2 + ρgz₁ = p₂ + ρu₂²/2 + ρgz₂

比エネルギー形式：p/ρ + u²/2 + gz = 一定　[J/kg]　…(4.7)
ヘッド形式：p/(ρg) + u²/(2g) + z = 一定　[m]　…(4.9)
記号説明単位
p圧力（静圧）Pa
ρu²/2動圧（運動エネルギー密度）Pa
ρgz位置圧（位置エネルギー密度）Pa
u流速m/s
z基準面からの高さm

導出のポイント

流体の単位体積当たりエネルギーを保存。
運動エネルギー ½ρu²（①）＋ 位置エネルギー ρgz（②）＋ 圧力による仕事 pAuΔt（③）の和が一定。
連続の式 A₁u₁ = A₂u₂ を使って AuΔt で割ると整理される。

⚠️ 適用条件：①定常流　②非圧縮性（密度一定）　③粘性なし（理想流体）　④同一流線上

例題（縮小管、d₁=80mm → d₂=40mm、h=100mm）

まず連続式で u₂ を求める：u₂ = (80/40)² × 6 = 24 m/s
①を基準面（z₁=0, z₂=h）にベルヌーイ適用：
p₂ = p₁ + ρ(u₁²−u₂²)/2 − ρgh
= 500×10³ + 1000(36−576)/2 − 1000×9.81×0.1 = 229 kPa

トリチェリーの定理 重要
u₂ = √(2gH)　…(4.16)
記号説明
u₂小孔からの噴出速度 [m/s]
H液面から小孔までの高さ [m]

導出

タンク液面①（u₁≈0、p₁=P₀）と小孔出口②（p₂=P₀）にベルヌーイを適用：
P₀ + ρgH = P₀ + ρu₂²/2
→ u₂² = 2gH　→　u₂ = √(2gH)
※タンク断面積が大きく水面速度 u₁ ≈ 0 と仮定。

例題

H = 2m のとき：u₂ = √(2 × 9.81 × 2) = 6.26 m/s

ピエゾメータ（圧力測定管）
p = P₀ + ρgH₂
H₂ = (p − P₀) / (ρg)

ポイント

ピエゾメータ（管壁に穴を開けた静圧測定管）の液面は静止しているので、液面高さは静水圧の式で決まる。
