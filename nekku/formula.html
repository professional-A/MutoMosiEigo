第1章

流体の基本性質

密度・比重量・比体積・比重 基本
ρ = m / V　　[kg/m³]
γ = W / V = ρg　[N/m³]（比重量）
v = 1/ρ　　　　[m³/kg]（比体積）
s = ρ / ρw　　（比重、無次元）
記号説明単位
ρ密度kg/m³
γ比重量（単位体積あたりの重量）N/m³
s比重（水との密度比）無次元
ニュートンの粘性法則 重要
τ = μ (du/dy)　[Pa]
一定勾配：τ = μ U/Y　　　せん断力：F = μA (U/Y)
両面せん断：F = μAU (1/Y₁ + 1/Y₂)
記号説明単位
τせん断応力Pa
μ粘性係数（粘度）Pa·s
U上板の移動速度m/s
Y板間距離m

導出のポイント

板間の速度分布は線形（上図）→ 速度勾配 du/dy = U/Y = 一定。
両面せん断は Y₁・Y₂ それぞれのせん断力を足す。引く力が最小になる板の位置は中央（x = h/2）。

例題（傾斜面を滑落する物体の最大速度）

油膜（厚さ δ）上を質量 m・傾斜角 θ の面を滑落：
重力とせん断力のつり合い：mg sinθ = μA(Umax/δ)
→ Umax = (δ · m · g · sinθ) / (μ · A)

動粘度
ν = μ / ρ　　[m²/s]
第2章①

静水圧・パスカルの原理

圧力の定義
P = F / A　[Pa = N/m²]
円形断面：P = 4F / (πd²)
パスカルの原理 重要
F₁/A₁ = F₂/A₂ = P
直径表記：F₂ = (d₂/d₁)² × F₁

ポイント

密閉容器内の静止流体に加えた圧力はあらゆる方向に等しく伝わる。
面積比が大きいほど力が増幅（油圧ジャッキの原理）。
レバー付きの場合：まず f × L₁ = F₁ × L₂ でF₁を求め、次に F₂ = (d₂/d₁)²F₁。

静水圧公式 重要
P = P₀ + ρgh　　[Pa]　…(2.9)
ゲージ圧：p = ρgh　[Pa]　…(2.10)
圧力ヘッド：h = p/(ρg)　[m]　…(2.11)
記号説明
P₀大気圧（絶対圧の基準）
pゲージ圧（大気圧基準）
h液面からの深さ [m]

導出のポイント

深さ h の微小要素に作用する力のつり合い → 上の液体の重さが加わる → p = ρgh（深さに比例）。

第2章②

液柱計（マノメータ）

💡 基本原理：同一水平面上の圧力は等しい。基準面 A での左右圧力を等値して解く。
U字管マノメータ 頻出
P₁−P₂ = ρ₂gy₂ + ρ'gh − ρ₁gy₁
同液体(ρ₁=ρ₂=ρ)：P₁−P₂ = ρg(y₂−y₁) + ρ'gh

導出（A面で等値）

左：P_A = P₁ + ρ₁gy₁
右：P_A = P₂ + ρ₂gy₂ + ρ'gh

逆U字管マノメータ
P₁−P₂ = ρ₁gy₁ − ρ'gh − ρ₂gy₂
ρ' < ρ（軽い流体が上部に）

導出（符号注意）

上部の軽い流体(ρ')が圧力を下げる方向に働く → ρ'gh の符号がマイナス。

単純マノメータ
P = P₀ + ρ'gh − ρgy　…(2.13)
記号説明
ρ'細管内液体の密度
ρ容器内液体の密度
h細管内液面の高さ
y基準面から容器液面までの距離
第2章③

平面板・曲面板に作用する力

平面板への全静水圧力 重要
F = ρg hG A　　[N]　…(2.21)
傾斜面：hG = yG sinθ　→　yG = hG/sinθ
記号説明
hG水面から図心 G までの鉛直深さ
A板の面積
yGO から G までの斜面距離
θ板と水平面のなす角

導出のポイント

合力 = 各微小面積に作用する圧力 p = ρgh の積分 → 結局 図心での圧力 × 全面積。

圧力中心（力の作用点） 重要
yC = IG/(yG·A) + yG　…(2.25)
円（直径d）：IG=πd⁴/64　→　yC = d²/(16yG) + yG
長方形（a×b）：IG=ab³/12　→　yC = b²/(12yG) + yG
圧力中心 C は常に図心 G より深い側（yC > yG）。深いほど圧力が大きいため、合力の作用点は下にずれる。

例題（傾斜円板 θ=36°、d=1.8m、上端深さ h=0.75m）

hG = 0.75 + (1.8/2)×sin36° = 1.279 m
yG = 1.279/sin36° = 2.176 m
F = ρg·hG·A = 31.9 kN
yC = 1.8²/(16×2.176) + 2.176 = 2.269 m

曲面板に作用する力 重要
Fx = ρg z̄ Ax　（水平分力）　…(2.26)
Fy = ρg V　　　（鉛直分力）　…(2.27)
F = √(Fx²+Fy²)　…(2.28)　　α = tan⁻¹(Fy/Fx)　…(2.29)
記号説明
Ax曲面の水平投影面積（鉛直面）
z̄その投影面の図心深さ
V曲面上部の液体体積

導出のポイント

水平分力 Fx：曲面を横から投影した面積への静水圧と同じ。
鉛直分力 Fy：曲面の上に乗っている液体の重量（浮力の反作用）。

例題（1/4円筒 r=1.2m、幅 b=4m）

Fx = ρg(r/2)·br = 28.25 kN
Fy = ρg(πr²/4)·b = 44.38 kN
F = 52.6 kN、α = 57.5°

第2章④

浮力（アルキメデスの原理）

浮力 重要
F = ρ流体 · g · V排除　…(2.30)
記号説明
ρ流体液体の密度（物体の密度ではない）
V排除液体中に沈んでいる部分の体積

重量 = 浮力のつり合い

浮く条件：ρ物体Vg = ρ流体V沈g
直方体（a×b）が浮く深さ：h = m/(ρ流体·a·b)

第3章

連続の式（質量保存）

連続の式 重要
A₁u₁ = A₂u₂ = Q　…(3.7)
直径表記：u₂ = (d₁/d₂)² × u₁
Q = Au = πd²u/4　[m³/s]　　G = ρQ　[kg/s]

導出のポイント

非圧縮性（密度一定）の定常流では、単位時間に通過する体積 Q は変わらない（質量保存）。
断面積↓ → 流速↑

例題

d₁=80mm → d₂=40mm、u₁=6 m/s：
u₂ = (80/40)² × 6 = 24 m/s

速度分布と平均流速 積分
【円管 — 放物線分布】
u = umax(1 − r²/R²)　　ua = umax/2　　…(3.8)
【平板間 — 放物線分布】
v = 2vmax(y/Y)(1 − y/Y)　　Q = bvmaxY/3

円管の積分

微小環状面積 dA = 2πr dr で積分：
Q = ∫₀ᴿ 2πr·umax(1−r²/R²)dr = πR²umax/2
→ ua = Q/A = umax/2

分岐管（Y字管）

Q₁ = Q₂ + Q₃
タンク充填時間：T = (D/d)²·(h/u)

第4章

ベルヌーイの定理とその応用

ベルヌーイの定理 最重要
p + ρu²/2 + ρgz = const　[Pa]　…(4.8)
p/ρ + u²/2 + gz = const　[J/kg]　…(4.7)
p/(ρg) + u²/(2g) + z = const　[m]　…(4.9)
2点間：p₁ + ρu₁²/2 + ρgz₁ = p₂ + ρu₂²/2 + ρgz₂
⚠️ 適用条件：①定常流　②非圧縮性（密度一定）　③粘性なし（理想流体）　④同一流線上

導出のポイント

単位体積あたりのエネルギー保存：
動圧 ½ρu² ＋ 位置圧 ρgz ＋ 静圧 p = 一定
連続式で u を消去して2点間の圧力差を求める。

例題（縮小管 d₁=80mm→d₂=40mm、p₁=500kPa）

u₂ = (80/40)²×6 = 24 m/s
p₂ = p₁ + ρ(u₁²−u₂²)/2 − ρgh = 229 kPa

トリチェリーの定理 重要
u₂ = √(2gH)　…(4.16)

導出

①（液面）と②（小孔）にベルヌーイ適用（どちらも P₀、u₁≈0）：
P₀ + ρgH = P₀ + ρu₂²/2　→　u₂ = √(2gH)

ピエゾメータ
p = P₀ + ρgH₂
H₂ = (p − P₀) / (ρg)

ポイント

管壁に細管を立てると液面が静止し、静水圧の式で圧力を読める。

v = 2v_max(y/Y)(1−y/Y) 平板間速度分布
