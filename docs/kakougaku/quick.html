S⚒ スプリングバック（超頻出）

スプリングバック ∝ σy / E
  • E（縦弾性係数）が小さい → スプリングバック大
  • σy（降伏応力）が大きい → スプリングバック大
  • ハイテン（高張力鋼）→ σy大 → 問題大
  • 対策：オーバーベンド・ボトミング・サーボプレス
真ひずみ ε = ln(l/l₀)　　体積一定：ε₁+ε₂+ε₃=0
トレスカ：τmax = σy/2　　ミーゼス：実験値に近い

S⚒ 加工温度区分

区分温度特徴
熱間再結晶以上変形抵抗小・加工硬化なし・スケール
温間前後中間・Mg/Ti向き
冷間再結晶以下高精度・加工硬化大・残留応力
  • Al再結晶≈150℃　Fe再結晶≈450〜600℃
  • FCC（Al・Cu）延性大◎　HCP（Mg・Ti）延性小→温間必要
主な加工法：
  • 圧延：V₀A₀=VrAr（体積保存）
  • 鍛造：ファイバーフロー→疲労強度最高
  • 転造：ネジ・歯車・加工硬化→疲労強度↑・材料ロスなし
  • 深絞り：LDR=D₀/d₁≈2.0〜2.2　しわ押えで割れ防止
  • せん断：クリアランス=板厚の5〜10%

名称説明
ビード溶接金属が凝固した部分
ボンド溶接金属と母材の境界
HAZ溶融しないが熱で組織変化した部分 → 破断起点
接合の分類：
  • 機械的：ボルト・リベット・SPR（異種材・孔なし）・焼きばめ
  • 冶金的（融接）：アーク・ガス・レーザー等
  • 冶金的（圧接）：スポット・FSW
  • ろう付け：母材溶かさない→熱影響小・異種材可

A🔗 FSW・ろう付け・検査

FSW（摩擦攪拌接合）
  • 回転ツールの摩擦熱で固相（溶融なし）接合
  • ○ 熱ひずみ小・欠陥少・Al/Mg◎
  • 700系新幹線床材・自動車・造船
  • 1991年英国TWI開発
ろう付け
  • 母材溶かさずろう材で接合
  • 硬ろう：融点 ≥ 450℃（銀ろう・銅ろう）
  • 軟ろう：融点 < 450℃（はんだ）→電子部品
非破壊検査
検査内/表ポイント
RT（放射線）内部引け巣・ブローホール
UT（超音波）内部厚板・現場可
PT（浸透）表面非磁性体もOK
MT（磁粉）表面磁性体のみ（Al不可）
ECT（渦電流）表面導電体・高速

A🔗 溶接欠陥と変形

欠陥原因・ポイント
ポロシティガス孔（H₂・CO）→脱ガス・乾燥
低温割れ水素＋マルテンサイト→予熱・後熱（PWHT）
アンダーカット電流過大→母材が溝状に
溶込み不良入熱不足→強度大幅低下
  • 変形メカニズム：加熱→膨張しようとする→拘束→塑性変形→冷却収縮→残留応力・変形
  • PWHT：550〜650℃→残留応力低減・靱性回復・H₂逸散
  • 引張残留応力→疲労亀裂進展→ショットピーニングで圧縮化
