1. 鋳造 (Casting)
概要・特徴・溶湯性質
  • 溶融金属→鋳型→凝固→製品
  • 複雑形状・中空形状 得意 / 精度△ 表面粗さ△
  • 融点↓の利点：金型使用可・省エネ・溶解容易
  • 複雑形状に適す理由：表面張力小→細部まで溶湯流入
  • 砂型に溶湯侵入しない理由：表面張力大→砂の隙間に入らない
  • 動粘度：流動性に影響（小→充填容易）
凝固メカニズム
  • 外周から冷却→核生成→デンドライト（樹枝状）成長
  • デンドライト間偏析：合金元素が固液界面に濃縮→成分不均一
  • 共晶（eutectic）：2成分が同時凝固→共晶点で凝固温度一定
  • 包晶（peritectic）：固相+液相→別の固相へ変態
  • 一方向凝固：タービンブレード向け→デンドライトを一方向に揃える
  • 肉厚薄い→冷却速度大→結晶粒細か→強度高

鋳造設計・検査
  • 抜き勾配（draft）：1〜3°、砂型から模型取出し
  • 角のR（丸み）：応力集中緩和→割れ・ひずみ防止
  • 見切り面：最も広い断面積（最大投影面）位置
  • 肉厚均一化：不均一→引け巣・残留応力の原因
  • アンダーカット：避ける（型が外れない）
  • 強度試験：同一溶湯で試験片別途鋳造→引張試験
  • 内部欠陥：超音波（UT）・放射線（RT）
  • 表面欠陥：浸透（PT）・磁粉（MT）
2. 塑性加工 (Plastic Forming)
概要・特徴
  • 固体のまま塑性変形で成形→歩留まり◎
  • 加工硬化で強度向上、量産に適す
  • 被加工材：塑性変形量大・加工硬化小
  • 金型材：耐摩耗性・耐熱性・耐食性 高
加工硬化メカニズム★
  • 塑性変形→転位（dislocation）密度増加
  • 転位同士が絡み合い→移動困難→降伏応力↑
  • 結果：強度・硬さ↑ / 延性・靭性↓
  • 冪乗硬化則：σ = C × εn
  • n値（加工硬化指数）：大→均一変形→深絞り向き
  • 焼なまし（annealing）：再加熱→転位消滅→軟化・延性回復
再結晶
  • 再結晶温度 ≈ 0.4×融点（K）、鉄は約450℃
  • 加工後に再結晶温度以上加熱→転位消滅・新結晶粒生成
  • 熱間加工：加工中に再結晶→変形抵抗低・大変形可
  • 冷間加工：再結晶なし→加工硬化蓄積
  • 加工度大+加熱温度高→再結晶粒径 小（強度高）
n値とr値★
  • n値（加工硬化指数）：σ=Cεⁿ のn。大→均一変形→深絞り割れにくい
  • r値（ランクフォード値）：r=ε幅/ε厚。板厚方向変形抵抗の指標
  • r値大→板厚方向に変形しにくい→深絞り限界↑（フランジ変形しやすい）
  • r値の面内異方性→耳割れの原因（Δr = (r₀+r₉₀-2r₄₅)/2）

  • 前進スリップ：出口材料速度＞ロール表面速度

塑性加工 計算・解析
ひずみ計算★
公称ひずみ e =ΔL/L₀ = (L-L₀)/L₀
真ひずみ ε =ln(L/L₀) = ln(1+e)
関係：ε < e（常に真ひずみ < 公称）
体積ひずみ：ε₁+ε₂+ε₃ = 0（非圧縮性）
  • L₀=100, L=150 → e=0.5, ε=ln(1.5)≈0.405
  • L₀=100, L=105 → e=0.05, ε=ln(1.05)≈0.0488
せん断加工荷重★
P =τs × L × t [N]
τs =せん断抵抗[MPa=N/mm²]
L =切断線長さ[mm] / t=板厚[mm]
  • 円形d=100mm, t=5mm, τs=300MPa
  • L=π×100=314.2mm
  • P=300×314.2×5=471,239N≈471kN
圧延 体積保存★
幅不変ならt₀·v₀ = t₁·v₁
→v₁ = v₀·(t₀/t₁) > v₀
一般：t₀·w₀·v₀ = t₁·w₁·v₁
  • t₀=10,v₀=2m/s, t₁=5mm → v₁=2×(10/5)=4m/s
  • t₀=10,v₀=1m/s, t₁=5mm → v₁=2m/s
降伏条件★
Tresca：σ₁ - σ₃ = Y
（最大－最小主応力＝降伏応力）
Von Mises：(σ₁-σ₂)²+(σ₂-σ₃)²+(σ₃-σ₁)² = 2Y²
2軸(σ₃=0)：σ₁²-σ₁σ₂+σ₂² = Y²
シミュレーション：主にVon Mises使用
  • Tresca：安全側（降伏しやすい）/ VM：実験値に近い
深絞り指標
LDR =D₀/Dp (限界絞り比, 大→深絞り性◎)
n値大：均一変形→割れにくい
r値大：板厚変形抵抗大→LDR↑
重要公式 まとめ
公称ひずみe = ΔL/L₀
真ひずみε = ln(L/L₀) = ln(1+e)
せん断荷重P = τs·L·t [N]

  • Net Shape：後加工不要で完成形→鋳造・塑性加工向き

溶接部の構造・欠陥・変形
溶接部断面（HAZ）
HAZ 相変態と予熱★
  • 溶接熱→HAZがオーステナイト化→急冷→マルテンサイト変態
  • マルテンサイト化→硬化・延性↓・低温割れ（水素割れ）
  • 炭素当量 Ceq（★要記憶）：
Ceq =C + Mn/6 + Si/24 + Ni/40 + Cr/5 + Mo/4 + V/14
Ceq > 0.44 → 予熱必要（50〜200℃）
目的：冷却速度↓→マルテンサイト変態抑制→割れ防止
  • 低温割れ：溶接後数時間〜24時間後に発生することあり
  • 予熱+後熱処理（SR焼なまし）で残留応力も低減
  • レーザ・FSWはHAZ最小（熱影響が狭い）
熱入力Q★
Q =（I×V×60）/ (v×1000) [kJ/mm]
I：電流[A] V：電圧[V] v：溶接速度[mm/min]
Q大：HAZ広・入熱大・冷却遅→溶接歪大
Q小：HAZ狭・急冷→割れリスク↑（高Ceq材）

FSW（摩擦攪拌溶接）★
  • 開発：1991年英国TWI
  • 適用：Al・Mg合金（航空・鉄道・造船・自動車）
  • メリット：①溶融欠陥なし②HAZ最小③残留応力小④省エネ
  • デメリット：①複雑形状不可②ツール穴（終端）③ツール消耗
  • 3パスで融接と同等強度に達する
疲労強度と継手効率
  • 余盛（ビード盛り）：断面急変→応力集中→疲労強度↓
  • アンダーカット：断面積減少→疲労破壊起点
  • 継手効率：溶接継手強度/母材強度。完全溶込み突合せで最高
  • 溶接部の残留引張応力は疲労強度を低下させる
  • ショットピーニングで表面圧縮残留応力→疲労強度向上

ろう接 詳細★
  • 硬ろう（Brazing）：>450℃ / Agろう・Cuろう・Alろう
  • 軟ろう（Soldering）：<450℃ / Sn-Ag-Cu（無鉛）・Pb-Sn（有鉛）
  • フラックス：酸化膜除去・ぬれ性向上（必須）
  • 毛管現象：接合隙間 0.05〜0.25mm が最適
  • 特徴：母材溶融なし・異種材料可・精密部品向き
  • 用途：電子基板はんだ付・銅管継手・超硬工具台金・タービン冷却穴
★ 試験の急所（頻出まとめ）
  • 砂型：表面張力が小さいから複雑形状OK / 砂の隙間は大きいので入らない
  • 融点↓の利点：金型可・省エネ・溶解容易（強度向上ではない！）
  • 引け巣：凝固収縮→内部空洞 / 対策：押湯 / 検査：超音波UT
  • 抜き勾配：砂型から模型を取り出しやすくするため
  • 角のR：応力集中緩和→ひずみ・割れ防止
  • 見切り面：最も広い断面積（最大投影面積）位置
  • 強度試験：同一溶湯から試験片別途鋳造→引張試験
  • 遠心鋳造：中子なしで円管→遠心力で内空形成
  • ロストワックス：耐熱超合金精密部品・タービンブレード
  • 加工硬化：転位密度増加→移動困難→強度↑延性↓
  • 冷間vs熱間：冷間=精度◎強化◎ / 熱間=大変形◎連続可
  • スプリングバック：縦弾性係数E大ほど大きい
  • 圧延v₁：t₀v₀=t₁v₁ → 出口速度大（体積保存）
  • ファインブランキング：逆力+反力+板押さえ同時→全面せん断面
  • MAG：Ar+CO₂ 鉄鋼向 / MIG：Arのみ 非鉄向 / MIG品質>MAG
  • TIG：タングステン非消耗電極（W）・高品質・薄板・SUS向き
  • サブマージド：粒状フラックス中・深溶込・熱効率◎・厚板・造船
  • EB溶接：真空中・深溶込・薄板ひずみ小・活性金属可・大型不可
  • HAZ：溶融しない組織変化領域 / 鋼→マルテンサイト化→硬化
  • 予熱の目的：Ceq高い鋼のマルテンサイト変態抑制→低温割れ防止
  • Ceq：C+Mn/6+Si/24+Ni/40+Cr/5+Mo/4+V/14 / >0.44で予熱必要
  • 溶接残留応力：局所加熱膨張→圧縮塑性変形→冷却収縮→引張残留応力
  • 溶接変形4形態：縦収縮・横収縮・回転変形・角変形
  • FSW：1991英TWI・固体状態・HAZ最小・Al/Mg合金
  • ろう接：硬ろう>450℃・軟ろう<450℃・母材溶融なし・異種材可
  • 内部NDT：UT（音波）・RT（X線） / 表面NDT：PT（毛管）・MT（磁束）
