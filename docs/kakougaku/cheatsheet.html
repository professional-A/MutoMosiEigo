加工学カンペ 2026前期中間｜鋳造編

■ 鋳造概要

  • 鋳造＝溶融金属を型に流し込み凝固させて成形
  • 砂型・金型・特殊鋳型の3種
  • Near Net Shape（ほぼ最終形状）→後加工少
  • 複雑形状・大型品に適す / Additive Manufacturingとも組合せ可

■ 鋳造材料選定条件（超重要）

①流動性：動粘度ν小→流れやすい→複雑形状充填可
②表面張力T大→砂の隙間に入らない（砂型向き）
③凝固収縮少→引け巣少
④割れが生じない（収縮応力に耐える）
溶湯特性表：
材料ν[cSt]T[N/m]流れ
鋳鉄0.7(大)≈1.87(大)流れにくい・砂型向き
銅0.41.36中間
Al0.2(小)0.91(小)流れやすい→複雑形状◎
・Alはν小→流れやすい→複雑形状充填に適す
・鋳鉄はT大→砂の隙間に侵入しない→砂型に適す

■ 砂型鋳造（各法の特徴）

生砂型

  • 砂＋ベントナイト（粘土）＋水で固める・繰返し使用可
  • ○コスト最低　×表面粗さ大・精度低
  • 大型品向き・ベント（穴）でガス抜き

自硬性砂型

  • 砂＋樹脂（フラン・フェノール等）＋硬化剤→常温硬化
  • ○中〜大型・精度◎　×再使用困難・コスト中

シェルモールド

  • 加熱金属型に樹脂コーティング砂を吹付→薄い殻型
  • ○表面良・精度高・中〜小型　×型費高・大型不可

フルモールド法（消失型鋳造）

  • 発泡スチロール模型を砂中に埋め→溶湯が模型を消失させ成形
  • ○アンダーカット可・中子不要・大型◎（100t maxも可）
  • ×ガス発生多・表面粗さ中

Vプロセス

  • 砂をプラスチックフィルムで密封し真空で保持
  • ○結合剤不要・表面良・繰返し可　×設備大型

ロストワックス（インベストメント）鋳造

  • ロウ模型→セラミックシェル形成→加熱でロウ除去→鋳造
  • ○最高精度・最良表面・複雑形状◎　×小型のみ・高コスト

■ 金型鋳造

重力金型鋳造

  • 重力で溶湯を金型に流し込む・砂型より精度◎
  • ×アンダーカット不可（中子必要）

ダイカスト（Die Casting）

  • 高圧（数十〜数百MPa）で金型に圧入充填
  • ○高精度・高速・薄肉可・大量生産◎（アルミ自動車部品等）
  • ×ガス巻込み（ブローホール）→熱処理困難
  • コールドチャンバー型：射出シリンダーが炉外→高融点合金（Al・Cu）向き
  • ホットチャンバー型：射出シリンダーが溶湯炉内→低融点合金（Zn・Pb）専用・高速

低圧鋳造

  • 炉に0.05〜0.15MPaかけ溶湯を押し上げ充填
  • ○引け巣少・精度中・Al製品向き

遠心鋳造

  • 回転型に溶湯を流し遠心力で充填凝固
  • ○円筒・管・パイプ専用・内側に中子不要
  • ○外側から凝固→引け巣少・組織緻密
  • ×円筒形状のみ

■ 鋳型設計ポイント

  • 見切り面：型の分割面→なるべく1平面に
  • 抜き勾配：型から抜けるよう1°以上の傾き
  • 角R（フィレット）：応力集中防止・割れ防止
  • 肉厚均一：局所的引け巣防止
  • アンダーカット：そのままでは抜けない形状→中子or消失型

加工学カンペ 2026前期中間｜塑性加工編

■ 塑性加工概要

  • 材料を降伏点以上の力で変形→永久ひずみ（塑性変形）
  • ○材料の無駄少・加工硬化で強度↑・大量生産◎・Near Net Shape
  • ×複雑形状難・残留応力・スプリングバック

■ 加工温度区分

区分温度特徴
熱間加工再結晶温度以上変形抵抗小・加工硬化なし（再結晶で回復）・酸化スケール発生
冷間加工再結晶温度以下（常温）高精度・高表面品質・加工硬化大・残留応力大
温間加工再結晶温度前後両者の中間・Mg・Ti等向き
  • Al再結晶温度：約150℃　Fe系：約450〜600℃　Ti：高温
  • FCC（Al・Cu・Au・Ni）：延性高・加工しやすい・すべり系多
  • BCC（Fe・Cr・Mo・W）：延性中・低温で脆性破壊注意
  • HCP（Mg・Zn・Ti・Co）：延性低・すべり系少→温間加工が必要

■ 主な塑性加工法

圧延（Rolling）

  • 2本以上のロール間で板・棒・線を延伸
  • 体積一定：V₀A₀ = VrAr（入側流速×断面積 = ロール周速×断面積 = 出側）
  • 板圧延・形鋼・線材圧延など広く使用

押出し（Extrusion）

  • ダイ孔から材料を押し出して断面形状を変える
  • 前方押出し（材料→ダイ方向）・後方押出し（ポンチ逆方向）
  • ○複雑断面形状◎・長尺製品・Al形材

引抜き（Drawing）

  • ダイ孔に材料を引き抜いて細く長くする
  • ○細線・パイプ・棒の精密加工　×加工度に限界（引張破断）

転造（Thread Rolling）

  • ダイを材料に押し当て転がしてネジ・歯車を成形
  • ○材料省略（切削なし）・加工硬化→疲労強度↑・高速量産

鍛造（Forging）

  • 型（金型）で材料を叩いたり押して成形
  • ○ファイバーフロー（流れ線）整列→疲労強度最高
  • 自由鍛造・型鍛造・冷間鍛造（Near Net Shape）

■ プレス機の種類

種類特徴
クランクプレス（機械式）高速・量産◎・ストローク固定・不意停止困難
油圧プレス大荷重・低速・ストローク可変・行程途中で停止可
サーボプレス速度・位置自由設定・精度最高・省エネ・高価

■ せん断加工（Shearing）

  • 材料をせん断力で切断（打抜き・切断・トリミング等）
  • クリアランス：上下刃の隙間＝板厚の5〜10%が適正
  • クリアランス大→ダレ大・破断面大・バリ大
  • クリアランス小→平滑せん断面↑・工具摩耗↑
せん断力： P = k × τ × l × t
（k:補正係数, τ:せん断強さ[MPa], l:切断周長[mm], t:板厚[mm]）

ファインブランキング

  • Vリング（板押え）＋超小クリアランス＋カウンターパンチ
  • ○全断面が平滑せん断面→高精度抜き◎（歯車・精密部品）

■ 曲げ加工（Bending）

  • V曲げ・L曲げ・U曲げ・ロール曲げ・ヘミング
スプリングバック（超重要）
曲げ後に弾性回復→目標角度より戻る現象
スプリングバック ∝ σy / E
→ E（縦弾性係数）が小さい → スプリングバック大
→ σy（降伏応力）が大きい → スプリングバック大
→ ハイテン（高張力鋼）は σy 大→スプリングバック問題大
→ 対策：オーバーベンド・ボトミング・形状補正・サーボ制御

■ 深絞り（Deep Drawing）

  • 平板からカップ形状を成形
  • LDR（限界絞り比）= D₀/d₁ ≈ 2.0〜2.2
  • ブランクホルダー（しわ押え）：フランジのしわ防止
  • ネッキング（くびれ）：引張応力集中→破断
  • バックアップ：ダイ肩R大・潤滑・材料延性高→LDR↑

■ しごき加工（Ironing）

  • 深絞り後の側壁を薄くする加工（アルミ缶・缶詰）
  • ○均一板厚・寸法精度↑

■ 降伏条件・塑性力学

トレスカ条件：
τmax = (σ1 - σ3)/2 = σy/2
→ 最大せん断応力が一定値(σy/2)で降伏

ミーゼス条件（Von Mises）：
(σ1-σ2)² + (σ2-σ3)² + (σ3-σ1)² = 2σy²
→ ひずみエネルギーが一定値で降伏
→ ミーゼスの方が実験値に近い
→ 等二軸引張：σ1=σ2、σ3=0 → ミーゼス: σy = σ1
トレスカ: σy = σ1（同じ）

■ 真ひずみ（対数ひずみ）

ε = ln(l/l₀) = ln(A₀/A)
体積一定：l₀A₀ = lA
工学ひずみ e = (l-l₀)/l₀ との違い：大変形では真ひずみ使用
ε₁ + ε₂ + ε₃ = 0（体積一定条件）

■ 加工硬化

  • 塑性変形→転位密度↑→材料が硬くなる
  • 冷間加工で顕著・熱間加工では再結晶で回復
  • 引張強さ・硬さ↑、延性↓

■ n値・r値（成形性指標）

n値（加工硬化指数）： σ = C × εⁿ
→ n大（0.2〜0.5）：加工硬化しやすい→くびれにくい→深絞り◎
→ n小（0.05〜0.1）：加工硬化少→局部的くびれ→成形性×

r値（ランクフォード値・塑性ひずみ比）： r = ε幅/ε厚
→ r > 1：板厚方向より幅方向に変形しやすい→深絞り性◎
→ r < 1：板厚が減りやすい→深絞り時にネッキング発生
→ 平均r値（r̄）大 → LDR大→限界絞り比向上

■ 主要材料と特徴

材料塑性加工での特徴
低炭素鋼(SPCC)延性高・深絞り◎・自動車外板
ハイテン(高張力鋼)強度高・軽量化・スプリングバック問題
アルミ合金(Al)軽量・FCC・深絞り可・スクラップ問題
Ti合金HCP・加工困難→温間or熱間・航空宇宙
Mg合金HCP・最軽量金属・温間加工必要
工具鋼(SKD/SKH)耐摩耗・プレス型材料

加工学カンペ 2026前期中間｜接合編

■ 接合の分類

機械的接合（Mechanical bonding）
→ ボルト・ナット / リベット / SPR / 焼きばめ / 巻き締め / 接着

冶金的接合（Metallurgical bonding）
→ 融接（溶融して接合）：アーク溶接・ガス溶接・レーザー等
→ 圧接（加圧して固相接合）：スポット・FSW・アプセット等
→ ろう付け（ろう材で接合）

リベット接合

  • 金属棒を孔に通し先端をかしめる→永久接合
  • ○構造物・航空機外板（昔）

SPR（Self Piercing Rivet）

  • 孔なしで打ち抜きながらかしめ接合
  • ○異種材接合（Al+鋼板）◎・自動車ボディ

焼きばめ

  • 外側を加熱膨張→軸に嵌め→冷却収縮で固定
  • ○高強度・ギヤ・ベアリング固定

巻き締め（Seaming）

  • 板端部を折り曲げて噛み合わせ（缶詰フタ）

■ 溶接部の名称（超重要）

・ビード（Bead）：溶接金属が凝固した部分
・ボンド（Bond）：溶接金属と母材の境界面
・HAZ（熱影響部 Heat Affected Zone）：溶融しないが熱で組織・性質が変化した部分
→ HAZは硬さ変化・靱性低下→破断しやすい領域

■ 各種溶接法

ガス溶接

  • 燃料：アセチレンガス炎（〜3500℃）で溶融接合
  • ○設備安価・可搬性高・現場作業可
  • ×大気遮蔽なし→酸化・低速・ひずみ大

被覆アーク溶接（SMAW）

  • 熱源：アーク放電　溶接棒：被覆剤（スラグでガスシールド）
  • ○設備安価・現場◎・オールポジション
  • ×低速・スラグ除去必要・溶接棒交換ロス

MAG溶接（Metal Active Gas）

  • シールドガス：Ar + CO₂（約80:20）
  • ワイヤ電極自動供給・連続溶接
  • ○高速・自動化◎・鉄系向き　×Al不可

MIG溶接（Metal Inert Gas）

  • シールドガス：Ar（不活性）
  • ○Al・ステンレス・非鉄金属向き

TIG溶接（Tungsten Inert Gas）

  • タングステン電極（非消耗）＋Arガスシールド
  • 溶加棒は別途供給（または不要）
  • ○薄板精密・美麗ビード・全材質可
  • ×低速（MIG > MAG > TIG の順で速度低下）

サブマージドアーク溶接（SAW）

  • 粉末フラックス中でアーク→大気を完全遮蔽
  • ○超高入熱・厚板◎・船舶・橋梁・高品質
  • ×下向き溶接のみ・設備大型・現場不可

エレクトロスラグ溶接

  • 溶融スラグ中のジュール熱でワイヤ溶融
  • 母材・当て金で冷却凝固・下から上に順次溶接
  • ○超厚板を1パス（1回）で溶接・電極数1〜数十本
  • △冷却が遅い→組織粗大化

レーザー溶接

  • CO₂レーザー・YAGレーザー・半導体レーザー
  • 薄板：表面加熱で溶融
  • 厚板：キーホール（孔）の周囲を溶融して接合
  • ○溶け込み深さ大・HAZ小・高速
  • ×装置高価　・銅は吸収率高→接合容易・Alは反射率高→接合しにくい

電子ビーム溶接

  • 電子ビームで母材溶融・真空チャンバー内で接合
  • ○接合部に熱集中・薄板でもひずみ小・溶け込み大
  • ○活性金属（Ti等）可　×大型部品不可・装置高価

■ 抵抗溶接（圧接）

スポット溶接（Spot Welding）

  • 電極で挟み大電流→ジュール熱で界面を溶融＋加圧
  • ○高速・自動化◎・自動車ボディ（1台あたり数千〜数万点）
  • ×点接合・外観に痕が残る

シーム溶接

  • ローラー電極で直線的に連続スポット溶接
  • 重ねた板→シーム / 突き合わせた板→バットシーム

アプセット溶接

  • 棒状材料を突き合わせ加圧＋ジュール熱→小径線材向き

フラッシュ溶接

  • 界面でアーク→表面を溶融→加圧接合・アプセットより大径向き

■ ろう付け（Brazing/Soldering）

  • 溶かした低融点ろう材で接合面を一体化
  • ○母材への熱影響小　○異種材料・複雑形状接合可
  • 硬ろう（Brazing）：融点450℃以上（銀ろう・銅ろう等）
  • 軟ろう（Soldering）：融点450℃以下（はんだ）→電子部品
  • フラックス：酸化膜除去→ぬれ性向上

■ 摩擦攪拌接合（FSW）

回転ツール（プローブ）を母材に押し込み摩擦熱で軟化→攪拌一体化
固相接合（溶融なし）→熱ひずみ・材質変化小・欠陥少
○Al・Mg等◎　○700系新幹線大型床材（1300W×5000L）
△複雑形状不可　△接合材固定必要
1991年英TWI開発→国内もTWIライセンスで使用

■ 溶接欠陥・変形・検査

欠陥原因・内容
ポロシティ溶接金属中のガス孔→強度低下
割れ（クラック）熱応力・収縮・水素脆化
アンダーカット母材が掘れた溝（電流過大）
溶込み不良母材が溶けていない→強度不足
変形（ひずみ）加熱冷却による収縮→設計余裕必要

検査法：RT・UT→内部欠陥 / PT（浸透）・MT（磁粉）・ECT（渦電流）→表面欠陥

■ 重要公式・数値まとめ

せん断力： P = k × τ × l × t
（k:補正係数, τ:せん断強さ, l:切断周長, t:板厚）

真ひずみ： ε = ln(l/l₀) = ln(A₀/A)
体積一定：ε₁ + ε₂ + ε₃ = 0　※常に ε < e（公称ひずみ）

スプリングバック ∝ σy / E
→ E小 or σy大 → スプリングバック大

LDR（限界絞り比）： D₀/d₁ ≈ 2.0〜2.2

加工硬化則： σ = C × εⁿ（n値大→成形性◎）
r値： r = ε幅/ε厚（r > 1 → 深絞り性◎）

トレスカ： τmax = (σ1-σ3)/2 = σy/2

ミーゼス： (σ1-σ2)²+(σ2-σ3)²+(σ3-σ1)² = 2σy²
（ミーゼスの方が実験値に近い）

圧延体積保存： V₀ × A₀ = Vr × Ar（ロール周速×断面積）

隅肉溶接： σ = P / (0.7h × l)
（h:脚長, l:溶接長, のど厚 = 0.7h）

PWHT（溶接後熱処理）： 550〜650℃ × 板厚25mmあたり1時間→残留応力低減・靱性回復

■ 鋳造 キーワード一覧

  • 流動性：ν(動粘度)小→流れやすい→複雑形状充填
  • 表面張力T大→砂の隙間に入らない（砂型○）
  • Al：ν小→複雑形状◎ / 鋳鉄：T大→砂型◎
  • 引け巣→押し湯・凝固順序制御・シミュレーション
  • ブローホール→脱ガス・乾燥・低速充填
  • ダイカスト→高圧充填・薄肉可・ガス混入注意
  • ロストワックス→最高精度・小型精密
  • 遠心鋳造→円筒・管・外側から凝固→緻密
  • フルモールド→消失型・アンダーカット可・大型
  • Near Net Shape→後加工最小化
  • 抜き勾配・見切り面・角R・肉厚均一→型設計基本

■ 塑性加工 キーワード一覧

  • 熱間：再結晶温度以上・変形抵抗小・スケール
  • 冷間：精度高・加工硬化大・残留応力
  • スプリングバック∝σy/E → E小・σy大で大
  • ハイテン→スプリングバック問題大
  • LDR≈2.0〜2.2（深絞り限界）
  • クリアランス→板厚の5〜10%
  • ファインブランキング→Vリング＋超小クリアランス
  • 転造→ネジ・歯車・材料省略・疲労強度↑
  • 鍛造→ファイバーフロー整列→疲労強度最高
  • トレスカ：τmax=σy/2 / ミーゼス：実験値に近い
  • 真ひずみ：ε=ln(l/l₀)
  • FCC(Al・Cu)延性高 / BCC(Fe)中 / HCP(Mg・Ti)低

■ 接合 キーワード一覧

  • HAZ（熱影響部）→溶融しないが組織・性質変化
  • ビード：溶接金属凝固部 / ボンド：母材との境界
  • MAG：Ar+CO₂→鉄系◎ / MIG：Ar→非鉄◎
  • TIG：W非消耗電極・薄板精密・全材質可
  • SAW：フラックス中アーク→厚板◎
  • エレクトロスラグ：ジュール熱→超厚板1パス
  • 電子ビーム：真空中・Ti等活性金属可
  • レーザー：キーホール→深溶込・HAZ小
  • FSW：固相接合・Al/Mg◎・熱ひずみ小・1991年
  • スポット：ジュール熱・自動車ボディ点接合
  • 硬ろう≥450℃（銀ろう・銅ろう）/ 軟ろう<450℃（はんだ）
  • RT・UT→内部 / PT・MT・ECT→表面
  • SPR→異種材（Al+鋼板）・孔なし接合
