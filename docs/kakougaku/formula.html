⚙️ 加工学 公式集

⚒ 塑性加工 — ひずみ・変形

真ひずみ（対数ひずみ） ε = ln(l / l₀) = ln(A₀ / A)
l:変形後長さ, l₀:元の長さ, A:断面積
工学ひずみ（公称ひずみ）との関係 e = (l − l₀) / l₀　→　ε = ln(1 + e)
大変形では常に ε < e（真ひずみの方が小さい）
体積一定条件 ε₁ + ε₂ + ε₃ = 0
3方向の真ひずみの和はゼロ
圧延・引抜きの体積保存（流量一定） V₀ A₀ = Vr Ar = V₁ A₁
V:速度, A:断面積（どの断面でも流量は等しい）
加工硬化則（n乗硬化則） σ = C · εⁿ
C:強度係数, n:加工硬化指数（n大→成形性◎・ネッキング起きにくい）
r値（ランクフォード値・塑性ひずみ比） r = ε幅 / ε厚
r > 1 → 板厚より幅が変形 → 深絞り性◎　/ 平均 r̄ 大 → LDR 大

⚒ 塑性加工 — プレス・成形

スプリングバック（超頻出） スプリングバック ∝ σy / E
E（縦弾性係数）小 → 大　/ σy（降伏応力）大 → 大
ハイテン（高張力鋼）は σy 大 → スプリングバック問題
せん断力 P = k · τ · l · t
k:補正係数（0.7〜1.0）, τ:せん断強さ[MPa], l:切断周長[mm], t:板厚[mm]
LDR（限界絞り比） LDR = D₀ / d₁ ≈ 2.0 〜 2.2
D₀:ブランク径, d₁:ポンチ径　/ これ以上絞ると破断
クリアランス（せん断加工） c = (5 〜 10%) × t
c:片側クリアランス, t:板厚 / 適正値で平滑断面・バリ最小
焼きばめの膨張量 Δd = α · d · ΔT
α:線膨張係数, d:内径, ΔT:加熱温度差 / Δd > しめしろ となる温度を選択

⚒ 塑性加工 — 降伏条件

トレスカの降伏条件 τmax = (σ₁ − σ₃) / 2 = σy / 2
最大せん断応力が σy/2 に達したとき降伏 / 計算が簡単
ミーゼスの降伏条件 (σ₁−σ₂)² + (σ₂−σ₃)² + (σ₃−σ₁)² = 2σy²
ひずみエネルギーが一定値で降伏 / 実験値に近い（ミーゼスの方が正確）
二軸引張（σ₁=σ₂=σ, σ₃=0）の比較 トレスカ: σy = σ₁　　ミーゼス: σy = σ₁（同じ）
純せん断（σ₁=τ, σ₂=0, σ₃=−τ） トレスカ: τy = σy / 2　　ミーゼス: τy = σy / √3
ミーゼスはトレスカより約15%大きい降伏応力を予測

🔗 接合 — 溶接の公式

ジュール熱（抵抗溶接・エレクトロスラグの基本） Q = I² R t
I:電流[A], R:抵抗[Ω], t:時間[s] / スポット溶接・エレクトロスラグに適用
入熱量（アーク溶接） Q = (V · I) / v　[J/mm]
V:アーク電圧[V], I:電流[A], v:溶接速度[mm/s]
Q大 → HAZ拡大・変形大 / Q小 → 溶込み不良リスク
隅肉溶接の応力 σ = P / (0.7h · l)
P:荷重[N], h:脚長[mm], l:溶接長[mm] / のど厚 a = 0.7h
のど厚（隅肉溶接） a = 0.7 h　（h:脚長）
等辺隅肉溶接の場合 / 有効のど断面 = a × l
PWHT（溶接後熱処理）条件 550 〜 650℃ × (板厚25mm あたり 1時間)
残留応力低減・HAZ靱性回復・水素逸散が目的

📌 覚える数値・条件一覧

鋳造
  • ダイカスト圧力：数十〜数百 MPa
  • 低圧鋳造圧力：0.05〜0.15 MPa
  • Al 動粘度 ν：0.2 cSt（最小）
  • 鋳鉄 表面張力 T：≈1.87 N/m（最大）
  • Al 再結晶温度：約 150℃
  • Fe 再結晶温度：約 450〜600℃
塑性加工
  • LDR（深絞り限界）：≈ 2.0〜2.2
  • クリアランス：板厚の 5〜10%
  • n値（高成形性）：0.2〜0.5
  • r値（深絞り性）：r > 1 で◎
  • スプリングバック：∝ σy / E
接合
  • 硬ろう融点：≥ 450℃（銀・銅ろう）
  • 軟ろう融点：< 450℃（はんだ）
  • PWHT温度：550〜650℃
  • 隅肉のど厚：a = 0.7h
  • FSW開発：1991年 英国 TWI
  • スポット電流：数千〜数万 A
